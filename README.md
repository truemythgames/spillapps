# spillapps

Monorepo for True Myth Games apps (API, CMS, Bible Tea mobile/web, shared packages).

## Structure

- `apps/api` — Cloudflare Workers API
- `apps/cms` — Content admin
- `apps/bible-tea` — Bible Tea product (mobile, web, content)
- `packages/db` — SQL migrations and seeds
- `scripts` — Content and codegen utilities

Private npm scope: `@spillapps/*`.
